# AmirAli Abedini — Personal Website

وب‌سایت شخصی و پورتفولیو AmirAli Abedini که به‌صورت استاتیک روی GitHub Pages میزبانی می‌شود.  
این مخزن شامل فایل‌های HTML/CSS/JS سایت، و فایل‌های کمکیِ SEO مثل `sitemap.xml` و `robots.txt` است.

---

## پیش‌نمایش (Live)
آدرس سایت (GitHub Pages):
`https://amiraliabedini110.github.io/`

## ساختار مخزن
- `index.html` — صفحهٔ اصلی  
- `blog.html`, `article-*.html` — صفحات مقالات  
- `/assets/` — تصاویر، CSS و JS  
- `sitemap.xml` — نقشهٔ سایت (برای موتورهای جستجو)  
- `robots.txt` — تنظیمات ربات‌ها  
- `README.md` — این فایل

## نحوه استقرار (Deploy) — سریع و عملی

2. فایل‌های جدید (مثل `sitemap.xml`, `robots.txt`, `README.md`) را اضافه کن:
